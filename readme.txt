Payport SP Parser v10.7

Требования:
- Windows 7/8/10/11
- Google Chrome должен быть установлен в стандартной папке
- Интернет соединение

Инструкция:
1. Запустите PP-Parser.exe
2. Убедитесь, что Chrome закрыт
3. Выберите нужные SP и настройте время
4. Нажмите "Запуск обработки"

Примечания:
- Программа автоматически запустит Chrome
- Отчеты сохраняются в папке REPORTS / reports_дата_время
